#pararius-scraper

Simple scraper targeting Dutch property search engine pararius.nl.

##Install

* ```git clone``` or donwnload this repository, ```cd``` to that folder
* ```npm install```
* ```cd /public```, ```bower install```

##Built with

###Server
* NodeJS
* Express
* request / cheerio

###Client
* jQuery
* Bootstrap / DynaTable
